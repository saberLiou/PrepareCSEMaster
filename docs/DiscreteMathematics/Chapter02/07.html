Theorem of Finite Set

∣A∣ |A| ∣A∣ = m m m < ∞ \infty ∞, ∣B∣ |B| ∣B∣ = n n n < ∞ \infty ∞,

  • f f f: A→B A \rightarrow B A→B 1-1 ⇒m≤n \Rightarrow m \le n ⇒m≤n
  • f f f: A→B A \rightarrow B A→B onto ⇒m≥n \Rightarrow m \ge n ⇒m≥n
  • f f f: A→B A \rightarrow B A→B 1-1 且 onto ⇒m \Rightarrow m ⇒m = n n n

Define Cardinality

A A A, B B B: sets, 若 ∃f \exists f ∃f: A→B A \rightarrow B A→B 為 1-1 且 onto, 稱 A A A 與 B B B 具相同之基數(cardinality): ∣A∣ |A| ∣A∣ = ∣B∣ |B| ∣B∣, 記作 A∼B A \sim B A∼B

∼ \sim ∼ 為一等價關係

Properties of Countable and Uncountable Set

  • 有限集 ⇒ \Rightarrow ⇒ 可數集(⇍ \nLeftarrow ⇍)
  • 不可數集 ⇒ \Rightarrow ⇒ 無限集(⇍ \nLeftarrow ⇍)
  • A A A, B B B: sets, A⊆B A \subseteq B A⊆B
    • B B B is countable ⇒A \Rightarrow A ⇒A is countable.
    • B B B is uncountable ⇒A \Rightarrow A ⇒A is uncountable.
