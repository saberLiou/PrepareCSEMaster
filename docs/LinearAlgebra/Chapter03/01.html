歐氏空間 (Euclidean space)

  • Define: V V V = Fn F^n F​n​​ = { (x1 x_1 x​1​​, x2 x_2 x​2​​, ..., xn x_n x​n​​) | x1 x_1 x​1​​, x2 x_2 x​2​​, ..., xn∈F x_n \in F x​n​​∈F }

    (x1 x_1 x​1​​, x2 x_2 x​2​​, ..., xn x_n x​n​​): n-tuple

  • V V V 上定義 + 及 ⋅ \cdot ⋅ : x x x = (x1 x_1 x​1​​, x2 x_2 x​2​​, ..., xn x_n x​n​​), y y y = (y1 y_1 y​1​​, y2 y_2 y​2​​, ..., yn y_n y​n​​), α∈F \alpha \in F α∈F
    • x x x + y y y = (x1 x_1 x​1​​ + y1 y_1 y​1​​, x2 x_2 x​2​​ + y2 y_2 y​2​​, ..., xn x_n x​n​​ + yn y_n y​n​​)
    • α⋅x \alpha \cdot x α⋅x = (αx1 \alpha x_1 αx​1​​, αx2 \alpha x_2 αx​2​​, ..., αxn \alpha x_n αx​n​​)
  • 當 F F F = R R R 或 C C C 時, 稱 V V V = Rn R^n R​n​​ 或 Cn C^n C​n​​ 為實或複歐式空間

矩陣空間 (matrix space)

  • Define: V V V = Fm×n F^{m \times n} F​m×n​​ = { A A A | A A A 為 m×n m \times n m×n matrix }
  • V V V 上定義 + 及 ⋅ \cdot ⋅ : 矩陣的加法與純量積
  • Fn F^n F​n​​ 有兩種觀點:
    • 行向量: Fn×1 F^{n \times 1} F​n×1​​
    • 列向量: F1×n F^{1 \times n} F​1×n​​
  • 當 F F F = R R R 或 C C C 時, 稱 V V V = Rm×n R^{m \times n} R​m×n​​ 或 Cm×n C^{m \times n} C​m×n​​ 為實或複矩陣空間

多項式空間 (polynomial space)

    P P P : 函數即向量，但不要試著根據其定義畫出空間的座標圖！

    • (f f f + g g g)(x x x) = f f f(x x x) + g g g(x x x)
